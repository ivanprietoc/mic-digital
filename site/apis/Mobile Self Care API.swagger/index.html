Mobile Self Care API

This is the standard API for Self Care functions related to Mobile Phones and all other Mobile Network services for Tigo. We use swagger-2.0 specification

This document is used mainly for new API design of new use cases. Traditional use cases have not been ported here yet. This is the reason we used v2

Version: 1.0.0

Terms of service:
